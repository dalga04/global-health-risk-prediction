# 🌍 Global Health & Lifestyle Risk Prediction

## 📌 Overview

This project aims to predict disease risk levels based on lifestyle factors using supervised machine learning.

## 🎯 Problem

Chronic diseases are heavily influenced by lifestyle habits such as:

* Physical inactivity
* Poor sleep
* Smoking
* Unhealthy diet

This project predicts health risk levels using data-driven techniques.

## 🧠 Approach

### 1. Data Preprocessing

* Handling missing values
* Outlier detection using IQR
* Feature scaling

### 2. Feature Engineering

* BMI Categories
* Activity Levels
* Health Risk Score

### 3. Model

* Random Forest Classifier

## 📊 Results

* Achieved good accuracy
* Key features:

  * BMI
  * Physical Activity
  * Sleep Duration
  * Smoking
